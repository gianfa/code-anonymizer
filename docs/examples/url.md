# Practical examples: URL

## Example 1: HTTP endpoint

Input:
```ts
const endpoint = "http://internal.service.local/api";
```

Output:
```ts
const endpoint = "https://example.com";
```

## Example 2: HTTPS endpoint

Input:
```ts
const webhook = "https://company.io/hooks/abc";
```

Output:
```ts
const webhook = "https://example.com";
```
